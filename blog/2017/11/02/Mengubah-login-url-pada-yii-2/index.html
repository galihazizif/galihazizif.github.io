Cara mengubah login url pada Yii2 Framework

Assalamu'laykum warakhmatullah, kali ini saya ingin menulis tentang cara mengubah url login pada yii2. Bagi para pengguna framework yii2 tentu sudah tidak asing lagi dengan fasilitas otentikasi user yang sudah disediakan secara built-in pada yii2.

Secara default, login url berada pada controller Site dan action login, namun ada kalanya kita harus mengganti halaman login tidak pada Site/login.

Untuk mengganti login url, kita cukup meng-override property loginUrl pada component user yang ada pada file konfigurasi. Jika pada file konfigurasi aplikasi Yii2 anda tidak terdapat component user maka perlu didefinisikan terlebih dahulu.

Setelah property loginUrl pada component user di override seperti konfigurasi diatas, halaman login akan diredirect ke url login yang baru.

Semoga bermanfaat.
